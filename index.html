Branches

São ramificações de um projeto de softwareou documento e tem como objetivo o desenvolvimento pararelo de:

  1. Novas funcionalidades;
  2. Correção de problemas;
  3. Alteração de código.

Branch Master ou Main

É o braço principal do seu projeto, do seu repositório e o que está sendo visto pelos usuários.

Branch Secundária

É o braço secundário do projeto, onde pode-se fazer alterações, remover ou incluir novas funções sem que atinja diretamente a Branch principal.

Hospedagem

O GitHub Oferece hospedagem gratuitas de Sites. Basta Criar uma conta e um repositório Público, e a URL será parecida com isso:


https:NomeUsuario.github.io/repositório
